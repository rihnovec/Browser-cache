Для работы проекта нужно:  
* Node.js версии 20 или выше;
* PHP 8.1;
* Composer версии 2;

Как запустить проект на локальном сервере:  
1. Настроить локальный сайт, адрес публичной части: папка public_html
2. В корне репозитория выполнить команды:
    * npm i
    * npm run build
    * composer install
